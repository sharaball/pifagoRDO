# pifagoRDO
zapret-discord-youtube от Flowseal патчнутый с большим количеством доменов для разблокировки и прочими вещами типа перевода service.bat на русский (и он **не** ломает **Rockstar** как zapret-discord-youtube от Flowseal с включенным Гейм фильтром)

README для zapret-discord-youtube [тут](https://github.com/Flowseal/zapret-discord-youtube/blob/main/README.md).


# Установка
1. нужно распоковать архив с pifagoRDO (он называется как pifagoRDOv0.0.0.zip) в любое место и **важно** чтобы по пути не было в названиях кириллицы (если тяжело то просто положите все файлы из архива в новую папку в корень диска C:/ с названием pifagoRDO)
2. включите SecureDNS и прочее что нужно **[см. README флоусила раздел установка](https://github.com/Flowseal/zapret-discord-youtube/blob/main/README.md#%EF%B8%8F%D0%B8%D1%81%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D0%B5)**
3. подбираем подходящюю вам стратегию (пробуйте все стратегии типа с ALT,TLS,FAKE итд) пока не найдем идеальную которая будет открывать все ваши нужные сайты
4. можно пользоватся!
5. если у вас не открываются .bat и другие файлы нажмите по файлу ПКМ > свойства и поставьте галочку Разблокировать и примените
